# Инструкция по созданию тестовой карты

## Способ 1: Использовать готовый GLB файл

1. **Скачайте готовый тестовый куб:**
   - Перейдите на https://threejs.org/
   - Найдите примеры GLB файлов
   - Или используйте любой готовый GLB файл

2. **Поместите в правильную папку:**
   ```bash
   client/public/models/map_test1.glb
   ```

## Способ 2: Создать карту в Blender

1. **Установите Blender** (если не установлен)
   - https://www.blender.org/download/

2. **Создайте простую сцену:**
   - Откройте Blender
   - Delete стандартный куб
   - Add → Mesh → Cube (для тестирования)
   - Add → Mesh → Plane (для пола) и масштабируйте (S + Z = 100)

3. **Добавьте материалы (опционально):**
   - Выберите объект
   - Перейдите в Shading
   - Добавьте простой материал

4. **Экспортируйте в GLB:**
   - File → Export → glTF Binary (.glb)
   - Выберите папку: `client/public/models/`
   - Назовите файл: `map_test1.glb`
   - Нажмите "Export glTF Binary"

## Способ 3: Создать карту программно (Node.js)

Используйте библиотеку `three` с `GLTFExporter`:

```bash
npm install three
```

Затем запустите скрипт для генерации карты.

## Текущее состояние

Просто поместите GLB файл в эту папку и перезагрузите страницу!
