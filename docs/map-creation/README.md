2. Подбираются текстуры и текстурирутеся. Важно отключать specular в шейдерах, чтобы по-умолчанию ничего не отражалось и т.д.
3. Когда уровень готов добавляет hdr карта
4. Далее работа со светом
5. Далее создание сущностей с Custom Properties которые определяют:

- Точки появления игроков - с Custom Property checkbox = respawn
